Mobile App Design

Description



This is a design for a transponder mobile app. The purpose of this app is to allow users to track a beacon or set up a beacon with their mobile device so that their friends and contacts may track it. Situations in which this could be useful would be at concerts or festivals, where it would help friends find each other and group up. Alternatively, the tracker could track an address and direct the user in the right direction.
